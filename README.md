# 📊 Mahindra Car Showroom Analysis

## 📌 Project Overview

This project analyzes Mahindra Car Showroom sales data to discover valuable business insights using Microsoft Excel.

The dashboard helps understand:

- Sales Performance
- Revenue Analysis
- Customer Trends
- Best Selling Cars
- Monthly Sales
- Regional Performance

---

# 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

---

# 📈 Dashboard Features

✔ Sales Overview

✔ Revenue Analysis

✔ Monthly Sales Trend

✔ Top Selling Models

✔ Customer Analysis

✔ Interactive Filters

---

# 📊 Business Insights

• Identified the highest selling car models.

• Compared monthly sales performance.

• Analyzed customer purchasing behavior.

• Discovered revenue contribution by different models.

• Built an interactive dashboard for business decision making.
